Introduction

Chat21 helps you to add real-time chat to your app. The iOS SDK provides you with methods to initialize and configure chat from the client side, and provides ready to use UI components to fast address the integration of a chat in your mobile App. Chat21 uses a backend totally written for the Firebase platform (http://firebase.google.com)
